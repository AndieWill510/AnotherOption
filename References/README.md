## Purpose

This folder holds reference stores and evidence maps for the book.

It separates:

```text
raw references
claim-control maps
reader-facing chapters and Options
```

A source appearing in this folder is not proof by itself.

The governing rule remains:

```text
Not what I can cite.
What I can prove.
```

See also: `../rfc/proof-not-citation-standard.md`.

## Drafting Gate

Before drafting or revising ABA / institutional-harm chapters or Options, check:

```text
References/ABA-EVIDENCE-MAP.md
```

If the claim is not mapped there, add or update the map before writing reader-facing prose.

Required workflow:

```text
References/REFERENCES.md -> evidence map -> claim review -> reader-facing chapter/Option
```

Do not move directly from a raw URL to prose.

---

## Standing Guardrails

- Do not overwrite existing files in this folder without explicit approval.
- Do not treat citation volume as proof.
- Do not let anger-era source dumps dictate the book’s current posture.
- Do preserve provenance, uncertainty, and gaps.
- Do narrow claims until they match the evidence.
