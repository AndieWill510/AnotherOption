Author: Kevin "Andie" Williams

Project: You Have Another Option  
Repository: AnotherOption  
Date: 2026-05-30

---

## 1. Purpose

This file preserves examples trimmed from Move sections.

The examples are useful, but too many examples inside one Option can create scan-load for a dysregulated reader.

Holding them here allows later reuse in:

- workbooks;
- companion guides;
- worksheets;
- later Options;
- accessibility aids;
- facilitator notes.

---

## 2. House Rule

Reader-facing Move sections should usually contain:

```text
one leading question or action
one to two examples
```

Five examples is a hard ceiling, not a target.

Trimmed examples should be held, not discarded.
